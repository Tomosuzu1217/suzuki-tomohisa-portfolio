# MENTAL SYNC - 内なる世界へダイブせよ (PoC)

https://mental-sync.web.app/

## 概要

最先端の**生成AI（Gemini 2.5 Flash）**と**心理学的アプローチ**を融合させた、自己理解と問題解決のための概念実証（PoC）アプリケーションです。
ユーザーの内面にある「感情」「価値観」「思考」をAIが分析・可視化し、**ポリアの問題解決法**に基づいた構造的なアプローチで、複雑な悩みの解決や目標達成を支援します。

> **Note**: 本アプリケーションは現在ローカルおよびFirebase HostingでのPoC（概念実証）段階です。

---

## ギャラリー

### 🧘‍♀️ 価値観ランタン (Value Lantern)
AIとの対話を通じて、ユーザーが大切にしている「価値観」を抽出し、ランタンとして可視化します。対話を重ねるごとにランタンが輝き、自己理解が深まります。

### 🧩 ポリアの問題解決 (Strategy Chat)
ジョージ・ポリアの「いかにして問題を解くか」に基づく4段階プロセス（理解→計画→実行→振り返り）を実装。AIがガイドとなり、漠然とした悩みを具体的なアクションプランへと導きます。

### 💬 感情AIチャット
ユーザーの感情（喜び、怒り、悲しみ、など）に寄り添うAIパートナーと、音声またはテキストで自然な会話が可能です。

## 主な機能

### 1. ポリアの問題解決法 (The Polya Method)
数学界の巨匠ジョージ・ポリアの問題解決フレームワークをAIアプリとして実装。
- **💡 理解する**: 現状とゴールを明確化
- **🗺️ 計画する**: 解決へのアプローチと障害（心配事）を洗い出し
- **🚀 やってみる**: 具体的なネクストアクションを決定
- **🔄 振り返る**: 結果を評価し、学びを蓄積
AIがファシリテーターとなり、初心者でも自然にこの思考プロセスを実践できます。

### 2. 価値観ランタン & 構造化
- **価値観抽出**: ユーザーとの対話から、背後にある価値観（Core Values）をAIが自動抽出。
- **3D可視化**: 抽出された価値観をランタンのようなオブジェクトとしてUI上に配置・整理。
- **構造化**: 価値観同士のつながりや、それに基づく行動指針を整理ボードで視覚化します。

### 3. マルチモーダル感情AI
- **音声認識 & 合成**: 自然な日本語での音声対話に対応。
- **感情分析**: ユーザーの入力から感情（怒り・喜び・悲しみ・楽しみ・秘めた思い）を分析し、最適な口調やアプローチで応答します。
- **性格プロファイル**: 会話履歴からユーザーの性格特性（Big Fiveなど）を推定し、接し方をパーソナライズします。

### 4. 分析レポート生成
- **ユーザー取扱説明書**: 蓄積されたデータ（価値観、行動パターン、性格特性）を基に、AIが詳細なレポートを生成。
- **PDF/CSV出力**: 分析結果をドキュメントとして保存可能。

## アプリケーション構造

```

├── app/                        # Next.js App Router
```

## 技術スタック

- **Frontend**: Next.js 14, React, TypeScript
- **UI & Styling**: Tailwind CSS, Framer Motion (アニメーション), Lucide React (アイコン)
- **AI Model**: Google Gemini 2.5 Flash (via Google AI Studio)
- **Backend / Auth**: Firebase Authentication (Google Auth), Firebase Hosting
- **State Management**: Zustand
- **3D Graphics**: React Three Fiber (R3F)
- **Voice**: Web Speech API (音声認識・合成)
