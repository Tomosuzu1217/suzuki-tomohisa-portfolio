# AIナレッジチャット - アプリケーションガイド

## 概要

**「あなたのファイルを、AIの知識に。」**

AIナレッジチャットは、PDF、Excel、Word、画像などのファイルをアップロードするだけで、その内容に基づいた回答を提供するRAG（Retrieval-Augmented Generation）システムです。強力な検索アルゴリズムとGoogle Gemini 2.5 Flashを組み合わせ、正確で信頼性の高い情報検索を実現します。

**[📱 アプリケーションを開く (https://rag-search-c85d8.web.app)](https://rag-search-c85d8.web.app)**

> **Status**: PoC (Proof of Concept) - 概念実証モデル

---

## 主な機能

### 🤖 高精度 RAG チャット
アップロードされた資料の内容をAIが理解し、質問に対して「資料のどこに書かれているか」を引用しながら回答します。

- **引用元の明示**: 回答の根拠となったファイル名と信頼度スコアを表示
- **ハルシネーション抑制**: 資料にない情報は「記載がありません」と正直に回答
- **日本語検索最適化**: TF-IDFとキーワードマッチングを組み合わせたハイブリッド検索

### 📂 マルチフォーマット対応
ビジネスで使われる主要なファイル形式にネイティブ対応しています。

| 形式 | 対応内容 |
|------|----------|
| **PDF** | 契約書、マニュアル、論文など |
| **Excel** (.xlsx) | 表データ、売上管理表など |
| **Word** (.docx) | 報告書、企画書など |
| **画像** (.png/.jpg) | スクリーンショットや写真から文字を読み取り (OCR) |
| **テキスト** (.txt) | メモ、ログファイルなど |

### 📊 管理者ダッシュボード
利用状況を可視化し、改善に役立つ分析データを提供します。

- **質問ログ分析**: どんな質問が多いかをランキング表示
- **感情分析**: ユーザーの満足度（ポジティブ/ネガティブ）を自動判定
- **共起ネットワーク**: 単語同士のつながりを可視化
- **CSV/Wordエクスポート**: 分析レポートをワンクリックでダウンロード

## 技術スタック (Tech Stack)

このアプリケーションは、スケーラビリティとコスト効率を意識したモダンな構成で構築されています。

### Frontend
- **Framework**: React, Vite
- **UI/UX**: Tailwind CSS, Lucide Icons, Framer Motion
- **Visualization**: Recharts (グラフ描画)

### Backend
- **Language**: Python 3.11
- **API Framework**: FastAPI
- **AI Model**: Google Gemini 2.5 Flash
- **Search Logic**: scikit-learn (TF-IDF), LangChain (Text Splitting)
- **OCR**: EasyOCR / PyDeduction

### Infrastructure (Serverless)
- **App Hosting**: Firebase Hosting
- **Backend Hosting**: Render (Web Service)
- **Database**: Firestore (Vector/Chunk Data)
- **Storage**: Firebase Storage (File Management)
- **Authentication**: Firebase Authentication

## 使い方ガイド

### 1. ログイン
- **ユーザー**: メールアドレスで新規登録・ログイン
- **管理者**: ログインパスワードに `admin-12171217Vv` を入力すると管理者モード有効化

### 2. ファイルのアップロード
- チャット画面左上のクリップアイコン📎またはドラッグ＆ドロップでファイルを追加
- 自動的にテキスト抽出・分割（チャンキング）が行われ、知識ベースに登録されます

### 3. 質問する
- 「大阪・天王寺の観光プランを教えて」のように自然言語で質問
- AIが関連する資料を探し出し、引用付きで回答します

### 4. 設定・管理
- **設定画面**: 検索の精度や回答の創造性（Temperature）をスライダーで調整可能
- **管理者ダッシュボード**: 統計情報の確認やログの管理が可能

## 開発のポイント

1. **コスト最適化**: 高価なVector DBを使用せず、TF-IDFとFirestoreの組み合わせで「完全無料」での運用を実現
2. **検索精度**: 日本語特有の分かち書きや言い回しに対応するため、独自の検索ロジックを実装
3. **UX**: 待ち時間を減らす工夫（楽観的UI更新）や、直感的なファイル管理インターフェース
