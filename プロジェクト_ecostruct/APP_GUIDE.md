# EcoStruct - ビジネス構造バイオミミクリー分析 (PoC)

https://business-analysis-e6e88.web.app/

## 概要

ビジネスの構造や課題を「自然界のメカニズム」になぞらえて分析・再構築する、バイオミミクリー（生物模倣）思考の概念実証（PoC）アプリケーションです。
最先端の生成AI（Gemini 2.5 Flash）を活用し、企業の有価証券報告書や事業データを読み込むことで、そのビジネスモデルを生態系や生物の器官として視覚化。論理的な分析だけでなく、直感的なアナロジー（類推）と多角的な視点（哲学・歴史・音楽など）を組み合わせることで、従来のフレームワークでは思いつかないような非連続なイノベーションを着想することを支援します。

> **Note**: 本アプリケーションは現在PoC（概念実証）段階です。

---

## ギャラリー

### 🧬 生体模倣バイオミミクリー分析
企業を一つの「生命体」や「生態系」として捉え直します。例えば、インフラ企業を「ローマ帝国」や「循環器系」に、スタートアップを「短命だが爆発的に増えるプランクトン」に例えることで、構造的な強みや弱みを直感的に理解できます。

### 🎹 タイムライン・サウンドスケープ
企業の「創業期」から「未来」までのライフサイクルを、視覚的なグラフと"音"で表現。成長率や安定性といったパラメータが、音楽のハーモニーやリズムの変化として生成され、企業の「鼓動」を聴覚的に体験できます。

## 主な機能

### 1. 構造的可視化 & インタラクティブグラフ
- **Biome Node Graph**: "Headquarters"（中枢神経）、"Factory"（消化器官）のように、ビジネスユニットを生物学的機能にマッピングしてグラフ化。
- **ドラッグ＆ドロップ探索**: 各ノードを操作しながら接続関係を確認し、複雑な依存関係を解きほぐします。

### 2. 多角的視点 & アナロジー生成
- **Multi-Perspective View**: 哲学、心理学、歴史、宗教、技術の5つのレンズを通して企業をプロファイリング。「存在意義」や「歴史的類似事例」を深掘りします。
- **Analogy Cards**: 「熱帯雨林のような多層構造」「渡り鳥のような分散ネットワーク」など、具体的なメタファーカードを生成。1文単位、あるいは「現象・構造・教訓」の単位でナレッジとして保存可能です。

### 3. ナレッジボード & アイデア創出
- **Polya's Heuristic Integration**: ポリアの問題解決法（分解、類推、一般化など）やMBAフレームワークを「知識チップ」として統合。
- **Dual Analysis Generator**: 収集したナレッジチップとユーザーの問いを掛け合わせ、論理的な「事実ベース分析」と、飛躍的な「メタファー分析」の2方向からアイデアを同時出力します。

### 4. 未来シナリオ & 合併シミュレーション (Merge Lab)
- **Scenario Planning**: 現在のパラメータから「7年後の未来」を予測し、楽観・悲観シナリオを提示。
- **Merge Lab**: 2つの異なる企業（例：トヨタ自動車 × Netflix）を入力し、その融合が生み出す化学反応や新サービスをAIがシミュレートします。

## アプリケーション構造

```
EcoStruct/
├── src/
│   ├── App.tsx                    # アプリケーションのエントリーポイント・ルーティング
│   ├── services/
│   │   ├── geminiService.ts       # Gemini API連携・プロンプトエンジニアリング（思考の核）
│   │   └── firebase.ts            # Firestore連携（認証・履歴保存）
│   ├── components/
│   │   ├── StructureGraph.tsx     # D3.js/SVGを用いたビジネス構造の可視化
│   │   ├── Soundscape.tsx         # Web Audio APIによるデータ可聴化・タイムライン表示
│   │   ├── KnowledgeBoard.tsx     # ナレッジ管理・アイデア生成ワークスペース
│   │   ├── AnalogyCard.tsx        # 細粒度選択可能なアナロジーカード
│   │   └── MultiPerspectiveView.tsx # 5つの観点による多層分析UI
│   ├── contexts/                  # 認証状態等のグローバル管理
│   └── types.ts                   # 型定義（バイオミミクリー概念のデータモデル化）
└── ...
```

## 技術スタック

- **Frontend**: React 18, TypeScript, Vite
- **UI Framework**: Tailwind CSS, Lucide React (Icons)
- **AI Model**: Google Gemini 2.5 Flash
- **Backend / Infrastructure**: Firebase (Authentication, Firestore, Hosting)
- **Audio Generating**: Web Audio API (リアルタイム波形生成)
- **Visualization**: Custom SVG & CSS Animations
